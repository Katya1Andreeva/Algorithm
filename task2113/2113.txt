Мы решаем задачу нахождения минимальной разницы высот между двумя точками в лабиринте,
которые могут быть достигнуты из начальной и конечной точек, соответственно.
Лабиринт представлен в виде двумерного массива, где каждая ячейка содержит значение высоты.

Алгоритм использует бинарный поиск для нахождения минимальной разницы высот,
при которой конечная точка достижима из начальной. Для этого используется функция solve,
которая проверяет, можно ли достичь конечной точки из начальной при заданной разнице высот.

Для поиска используется алгоритм поиска в ширину (BFS) с использованием очереди
deque из модуля collections. Алгоритм BFS проходит через все смежные точки,
начиная с начальной, и помечает их как посещенные, если они удовлетворяют определенным условиям.

После нахождения минимальной разницы высот выводится результат или -1,
если конечная точка недостижима.
